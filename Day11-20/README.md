## ✅ Day 11

### Problem: Addition of Two Matrices

**Description:**  
Write a C program to perform addition of two matrices having the same dimensions.  
Each element of the resulting matrix is obtained by adding the corresponding elements  
from the two input matrices.

**Concepts Used:**
- 2D Arrays
- Nested loops
- Matrix operations
- Basic input/output

**File:**  
`Day11.c`

## ✅ Day 12

### Problem: Check Whether a Matrix is Symmetric

**Description:**  
Write a C program to check whether a given matrix is symmetric.  
A matrix is symmetric if it is a square matrix and for every i and j,  
element[i][j] is equal to element[j][i].

**Concepts Used:**
- 2D Arrays
- Nested loops
- Matrix transpose concept
- Conditional statements
- Basic input/output

**File:**  
`Day12.c`
